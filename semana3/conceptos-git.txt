Repositorio:
Un repositorio es el espacio donde se almacenan los archivos de un proyecto y todo su historial de cambios. Permite organizar el trabajo y mantener un control sobre las modificaciones realizadas durante el desarrollo.

Commit:
Un commit es un registro o “captura” de los cambios realizados en un proyecto. Cada commit guarda información sobre las modificaciones hechas y permite volver a versiones anteriores si es necesario.

Push:
El comando push se utiliza para enviar los cambios del repositorio local hacia un repositorio remoto, como GitHub. Esto permite publicar el trabajo y compartirlo con otras personas.
